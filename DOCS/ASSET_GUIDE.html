General Specifications

All game assets sit on a 64×64 pixel tile grid. Units and buildings must fit within this grid. Player-owned units should be tintable to the player's colour (use white/grey base with a colour overlay, or design with a clear team-colour region).

Art Style Recommendations

  • Original game used pre-rendered 3D sprites in an isometric-ish top-down view
  • Modern approach: clean top-down pixel art or hand-painted style at 64×64
  • Each unit needs 8 directional frames (N, NE, E, SE, S, SW, W, NW) for rotation
  • Export as PNG with transparency
  • Use a consistent colour palette across all assets
  • Player colour should be a distinct region (e.g. hull stripe, banner) that can be recoloured

File Format & Sizes

  • Format: PNG with alpha channel
  • Standard unit: 64×64 px per frame
  • Big building (2×2 tiles): 128×128 px
  • Small building (1×1 tile): 64×64 px
  • Terrain tiles: 64×64 px, seamlessly tileable
  • Directional sprites: 512×64 px sprite sheet (8 frames) or individual files
  • Effects/FX: variable size, typically 64×64 to 128×128 per frame

Terrain Tiles

4 base types + variants
TileDescriptionSizeVariants NeededPriority
Ground Default land terrain. Earthy/dirt/grass mix. 64×64 4-8 variants for visual variety. Must tile seamlessly. HIGH
Water Deep water. Ships and subs operate here. 64×64 2-4 variants. Optional: 2-frame animation for gentle waves. HIGH
Coast Shore/beach between land and water. 64×64 8+ edge variants (transitions between water and land in all directions). HIGH
Blocked Impassable mountain/rock terrain. 64×64 4-8 variants. Rocky, mountainous look. HIGH
Note: The original game stored ~400 terrain tile graphics per map in the WRL map file. For the new version, a tileset of 30-50 well-designed tiles with auto-tiling rules will produce better results. Coast transitions are the most complex—consider using Godot's TileMapLayer auto-tile system.

Priority Summary — What to Create First

üî¥ HIGH Priority (Start Here)

  • 4 terrain tiles (Ground, Water, Coast, Blocked) — the map is the backdrop for everything
  • Constructor, Engineer, Surveyor, Tank — the 4 starting units every player gets
  • Mining Station, Small Generator, Power Station — core economy buildings
  • Connector — the most-placed building in every game
  • Explosion (Small + Big), Hit Spark — combat feels lifeless without these
  • Selection Ring, Movement Path, HP Bar — core UI feedback

üü° MEDIUM Priority (Round Two)

  • Remaining combat vehicles: Assault Gun, Scout, Mobile AA, Rocket Launcher, Missile Vehicle, Infantry
  • Naval units: Gunboat, Escort, Corvette
  • Aircraft: Fighter, Bomber
  • Production buildings: Light/Heavy Vehicle Plants, Aircraft/Ship Factories
  • Defence buildings: Gun Turret, AA Gun, Artillery, Missile Launcher
  • Economy: Storage buildings, Gold Refinery, Road, Bridge, Platform
  • Effects: Muzzle flashes, Rocket projectile, Smoke
  • UI: Cursors, Resource icons, Fog of War, Resource overlay

üü¢ LOW Priority (Polish)

  • Specialist vehicles: Commando, APC, Mine Layer, Scanner, Repair, Bulldozer, Transports
  • Remaining naval: Missile Ship, Submarine, Sea Transport, Cargo Ship, Sea Mine Layer
  • Remaining air: AWACS, Air Transport
  • All 4 alien units
  • Support buildings: Depot, Dock, Hangar, Barracks, Training, Landing Pad, etc.
  • Effects: Dark Smoke, Torpedo, Absorb, Vehicle Tracks
  • Misc: Rubble, Corpse, Minimap frame, Buttons
